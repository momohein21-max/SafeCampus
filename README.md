# SafeCampus Smart Support System

A campus safety application built using Java and gRPC.

## Services
- Incident Report Service (Client-Streaming)
- Smart Lighting Service (Unary)
- Safe Route Service (Server-Streaming + Bidirectional)

## How to Run
1. Run IncidentReportServer.java
2. Run SafeRouteServer.java
3. Run SmartLightingServer.java
4. Run ClientGUI.java

## Technologies
- Java
- gRPC
- Protocol Buffers
- JmDNS
- Maven
